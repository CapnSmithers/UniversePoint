Universe Point Ultimate Frisbee scorekeeping app
Developed by Matthew Smith
Copyright 2013

This app is intended for use as a scorekeeping application for ultimate frisbee.  Currently it is in the alpha
stage of development, but still provides basic functionality. The current scope of the app is below:

Main Menu Screen - Buttons to take you to scorekeeping screen (New Game) and leave the app (Exit)

Scorekeeping screen - Can edit team name, add/remove points and add/remove turnovers.  Finish Game button takes
you to the game summary screen

Summary Screee - Displays team name (edited on the scorekeeping screen), along with team score and number of 
turnovers.  Buttons allow you to score a new game or return to the main menu.
